Platform specific
  • Windows
    • Links with DbgHelp.dll.
  • GCC
    • Links with libbfd by default. Optional.

Define BOOST_CALL_STACK_NO_OPTIONAL_LIBS to cut dependencies on libraries that are not installed by default on the platform (optional libraries). This could affect the level of detail of the functionality of this library. By default, optional libraries are used and linked with.

For instance, this will cut the dependency on libbfd on GCC platforms and some symbols will not resolve properly, nor would line numbers be available.
